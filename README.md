# Wordle
The files must be kept in the same folder for both versions of Wordle! In the PySimpleGUI version, the main file pySimpleWordle.py uses logic from wordleFunctions.py.

## Text Version
- wordleText.py
- 5letterwords.txt

## PySimpleGUI Version
- pySimpleWordle.py 
- wordleFunctions.py
- 5letterwords.txt
